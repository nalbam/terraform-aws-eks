* <https://github.com/kubernetes-sigs/aws-iam-authenticator>

## Create Cluster

```bash
# terraform
brew install terraform

# examples
cd examples/dev-demo

# terraform (10m)
terraform init
terraform plan
terraform apply

# get
kubectl get node -o wide
kubectl get all --all-namespaces
```
